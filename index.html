The JavaScript Array object is a global object that is used in the construction of arrays; which are high-level, list-like objects.

map() Method

"The map() method creates a new array with the results of calling a provided function on every element in the calling array." 1

map() creates a new array after running a function on each element of the original array.

newArray = oldArray.map(element=>{

//code that transforms element

return transformed element;
});

priceOnlyArray is an array of raw prices. Using the map() method it calculates a new price by adding on the proper tax. It returns each new price to arrayWithTaxes. 


    const priceOnlyArray = [1.25, 2.00, 5.89, 12.30];
                
    const arrayWithTaxes = priceOnlyArray.map(price=>{
        //tax is 4.25%   
        const taxPercent = 0.0425;
        const priceWithTax = price + (price * taxPercent); 
        //round priceWithTax to 2 decimal places and convert to a number before returning
        return Number(priceWithTax.toFixed(2);
    });

    //arrayWithTaxes = [1.3, 2.08, 6.14, 12.82]

1 For more detailed information, visit the Mozilla Developer Network's entry on map().

reduce() Method

The reduce() method executes a reducer function (that you provide) on each member of the array resulting in a single output value.

slice() Method

The slice() method returns a shallow copy of a portion of an array into a new array object selected from begin to end (end not included). The original array will not be modified.

splice() Method

The splice() method changes the contents of an array by removing or replacing existing elements and/or adding new elements.

filter() Method

The filter() method creates a new array with all elements that pass the test implemented by the provided function.
