Arrays are useful for storing multiple elements of data in one place. The Array object has many built-in methods which make manipulting them simple.

map() Method

Definition

"The map() method creates a new array with the results of calling a provided function on every element in the calling array." 1

map() creates a new array after running a function on each element of the original array.

newArray = oldArray.map(element=>{
                
    //code that transforms element
            
    return transformed element;
});

Code Example

priceOnlyArray is an array of raw prices. Using the map() method it calculates a new price by adding on the proper tax. It returns each new price to arrayWithTaxes. 

const priceOnlyArray = [1.25, 2.00, 5.89, 12.30];
                
const arrayWithTaxes = priceOnlyArray.map(price=>{
    //tax is 4.25%   
    const taxPercent = 0.0425;
    const priceWithTax = price + (price * taxPercent); 
    //Round priceWithTax to 2 decimal places,
    //and convert to a number before returning.
    return Number(priceWithTax.toFixed(2);
});

//arrayWithTaxes = [1.3, 2.08, 6.14, 12.82]
1 For more detailed information, visit the Mozilla Developer Network's entry on map().
reduce() Method

Definition

"The reduce() method executes a reducer function (that you provide) on each member of the array resulting in a single output value." 1

reduce() takes an initial value, and the first element of the array, preforms an operation between the two, and returns the result as an accumulator value. Each element of the array is used to adjust the accumulator according to your code. Once every element has had its chance, the final accumulator value is returned to be used as a single value.

const reducedArrayOutput = array.reduce(accumulator, arrayElement=>{

    //code that takes each arrayElement and the accumulator,
    //preforms an operation between the two, and returns the result
    //as the accumulator for use in the next iteration 
        
    //initialValue is used as the first iteration's accumulator value,
    //or if initialValue is not included, the array's zero element is
    //the initialValue

    //once the last array element and the accumulator have been combined,
    //the resulting value is returned as reducedArrayOutput  

}, initialValue);

Code Example

januaryMileage is an array of miles driven in a day. reduce() will be run on each day in januaryMileage, multiply it by the average cost per mile, and then add it to the accumulator. costOfDrivingJanuary will equal the total miles in the array multipled by cost per mile.

const januaryMileage = [102, 55, 86, 275, 10, 92];

const reducer = function(accumulator, milesInADay){

    const costPerMile = 0.55;

    return accumulator + (milesInADay*costPerMile);
};

const costOfDrivingJanuary = januaryMileage.reduce(reducer, 0);

//costOfDrivingJanuary = 341;

slice() Method

Definition

"The slice() method returns a shallow copy of a portion of an array into a new array object selected from begin to end (end not included). The original array will not be modified." 1

slice() creates a new array from an exact portion of an old array. It takes two parameters, the first is index to begin extraction, and the second is to end extraction. If the second index is left out, it will give everything to the end. Remember that arrays begin at the zero index.

  • A negative index number counts backwards from the end of array.
    • slice(-3) extracts the last three elements
  • The end index is not included in the new returned array
    • slice(0, 2) will return the first (0) and second (1) elements, and not the third (2).
  • If no indexes are passed in at all, a complete copy of the old array is returned as a new array.
newArray = oldArray.slice(begin, end);

Code Example

studentsArray is an array of objects. Each object has a student's first name, and his/her birthday. The teacher wants to know only the first three students' information.

const studentsArray = [
    {name: "Dan", birthday: "01/02/12"},
    {name: "Sarah", birthday: "03/12/12"},
    {name: "Rich", birthday: "08/22/12"},
    {name: "Miguel", birthday: "10/10/12"},
    {name: "Michael", birthday: "09/23/12"},
    {name: "Clarissa", birthday: "12/24/12"}
];

const firstThree = studentsArray.slice(0,3);

/*
    [{name: "Dan", birthday: "01/02/12"},
    {name: "Sarah", birthday: "03/12/12"},
    {name: "Rich", birthday: "08/22/12"}]
*/

splice() Method

Definition

"The splice() method changes the contents of an array by removing or replacing existing elements and/or adding new elements." 1

splice() takes parameters that acutally change the array it is called upon. Useful for adding and removing elements from within an existing array.

  • First parameter: start
    • Index at which splice will begin.
    • If start is negative, splice will happen that many indices from the end.
  • Second parameter: deleteCount
    • Optional; you don't have to delete anything to use splice()
    • Number of elements to remove from the array, beginning at start
  • Third parameter: item
    • Element to be inserted into array.
    • Optional; nothing has to be added to the array to use splice()

You can call splice() to only add elements, only delete elements, or delete then replace the index with an item. splice() returns any elements that were deleted.

array.splice(start, deleteCount, item1, item2, item3);

//deleteCount and any item(s) are optional 
//deleted elements are returned

Code Example

A greenhouse keeps track of plant heights to determine when harvesting is necessary. The robot that scans the heights, missed two heights near the beginning. We'll use splice() to manually add the missing heights back into the plantHeights array.

const plantHeights = [10.1, 10, 10.4, 10.1, 10.2, 10.8, 10.9, 10.3];

//add two new heights after the second element

plantHeights.splice(2, 0, 10.1, 10.4);

//[10.1, 10, 10.1, 10.4, 10.4, 10.1, 10.2, 10.8, 10.9, 10.3]
1 For more detailed information, visit the Mozilla Developer Network's entry on splice().
filter() Method

Definition

"The filter() method creates a new array with all elements that pass the test implemented by the provided function." 1

filter() creates a new array by running a function once for each element in the old array. The function must return true or false. If the function returns true on an element, it will be included in the new array. 

const newArray = oldArray.filter(element=>{

    //if element passes test
        return true

    //if element cannot pass test
        return false
});

Code Example

A convenience store wants to find out what percent of its total sales for the day were greater than $20.00. By using filter(), the sales amounts listed in an array can be tested if they are greater than or equal to $20. largeSales is a new array created from the sales123118 array after running filter(). 

const sales123118 = [1.25, 1.00, 5.04, 3.20, 58.67,
                    8.90, 6.75, 20.15, 18.24, 11.24,
                    1.45, 3.76, 31.56, 10.00, 20.00, 2.54];

const largeSales = sales123118.filter(sale=>{

    if (sale >= 20){
        return true;
    } else {
        return false;
    }
});

//largeSales = [58.67, 20.15, 31.56, 20]
1 For more detailed information, visit the Mozilla Developer Network's entry on filter().
